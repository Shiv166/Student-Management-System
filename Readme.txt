Project Name: Student Management System
Author Name : Shivang Khatri

An end-to-end system in which front-end is designed using coding languages HTML, CSS and Bootstrap while maintaining database using coding languages such as PHP and MySQL. The system is designed to maintain scores, assignments, access attendance and real time college as well as university news.

Faculties will have option to add marks, add students, add study materials and view students that are currently present in the dashboard.

Upload sms sql file in phpmyadmin to get the database.
